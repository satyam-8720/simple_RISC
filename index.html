About SimpleRISC

SimpleRISC is a simplified RISC (Reduced Instruction Set Computer) architecture designed for educational purposes. This web-based assembler translates SimpleRISC assembly language into machine code.

Assembler Features

  • Two-pass assembly process for handling labels and symbol resolution
  • Support for all SimpleRISC instructions including arithmetic, logical, and control flow operations
  • Register and immediate addressing modes
  • 32-bit instruction encoding format

Instruction Format

SimpleRISC uses a 32-bit instruction word with the following general format:

  • Bits 27-31: 5-bit opcode field
  • Bit 26: Mode bit (0 for register operands, 1 for immediate operands)
  • Bits 21-25: First register operand (rd)
  • Bits 16-20: Second register operand (rs1)
  • Bits 11-15: Third register operand (rs2) or part of immediate value
  • Bits 0-10: Extended immediate field or unused
